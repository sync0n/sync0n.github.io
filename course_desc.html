Introduction

In this course you will get an introduction to the basic practices in the myKN environment, and develop skills you need as an HR professional.

You will learn how to use the tools we have to enhance your skills and experience. You will also explore the basic principles of communication, file usage and profile management on the platform.

Overview

Module 1 - Forums

In this module you will learn how to use forums. We will show how to find and follow forums or start your own.

Module 2 - Files

This module will teach you how to organize, upload and access files either from myKN or the IBM Connections desktop plugin.

Module 3 - Communication/Networking

This module focuses on communication and collaboration.
